## 概述

<span class='marker'>**Paradocs** 是一个面向段落的文档演示系统。</span>它适用于逐段展示大篇幅文本，同时进行评论和讲解。使用 Paradocs，每按一次按键或按钮，就会依次高亮显示一个句子。这使观众能够清楚地知道演示者当前聚焦在哪个句子上。

Paradocs 的创作者最初是为了个人在大学 ESL 阅读课上的教学需要而开发的。在阅读课上，教师和学生往往大部分时间都在低头看文本，他觉得这是一件令人遗憾的事情。<span class='marker'>使用 Paradocs，您只需将原始文本整理成"一行一句"的格式，就能轻松制作课堂演示文稿。</span>您还可以将其与 Zoom 等远程会议应用配合使用，进行在线教学。

<span class='marker'>Paradocs 具备多语言文本转语音（TTS）功能，</span>它利用 Google Chrome、Mozilla Firefox、Apple Safari 和 Microsoft Edge 等浏览器的 Web Speech API，允许您选择计算机上安装的多种语言之一，在演示过程中随时朗读当前句子。

演示文稿可由用户自行进行口头演示。<span class='marker'>此外，您也可以让系统自动进行整个演示。</span>点击屏幕左上角的魔法棒图标，从当前幻灯片的下一个片段到整个演示文稿的最后一个片段，文本朗读、视频播放等将自动依次执行。

Paradocs 的许多功能依赖于 Hakim El Hattab 开发的演示文稿创建库 [Reveal.js](https://revealjs.com)。在此衷心感谢他和他的合作者们提供了这个出色的库。精美的背景壁纸由 <a href='https://www.transparenttextures.com/'>Transparent Textures</a> 提供。

Paradocs 由 [Yoichiro Hasebe](https://yohasebe.com) 开发。所有文本处理完全在浏览器中完成——不会将任何数据发送到服务器。
