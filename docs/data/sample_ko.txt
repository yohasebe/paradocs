## Paradocs

| **단락 중심 문서 프레젠테이션 시스템**
----
Paradocs는 단락 중심 텍스트 프레젠테이션 시스템입니다.
대량의 텍스트 📃 를 단락별로 제시하면서 코멘트와 설명을 덧붙이는 데 적합합니다.
Paradocs에서는 키 ⌨️ 나 버튼 🖱 을 한 번 누를 때마다 문장이 차례로 하이라이트 🖍 됩니다.
이를 통해 청중은 발표자가 현재 어떤 문장에 주목하고 있는지 파악할 수 있습니다.
----
### 블록의 종류
----
#### 일반 단락

이것은 일반 단락의 텍스트입니다.
일반 단락에서는 키를 누르면 문장별로 하이라이트가 이동합니다.
일반 텍스트의 문장은 포커스를 받으면 하이라이트 🖍 됩니다.
포커스를 받지 않은 문장은 회색으로 표시됩니다.
텍스트가 슬라이드에 다 들어가지 않으면 자동으로 스크롤됩니다.

----
#### 정적 단락

아래는 정적 단락의 예시입니다.
정적 단락을 만들 때는 줄의 맨 앞에 세로줄 기호를 추가합니다.

| 이것은 정적 단락입니다.
| 정적 텍스트의 문장은 하이라이트되지 않습니다.
| 항상 검은색으로 표시됩니다.

----
#### 순서 없는 목록

순서 없는 목록이란 이른바 글머리 기호 ⚫ 형식을 말합니다.
순서 없는 목록의 항목을 기술하려면 각 줄의 맨 앞에 별표와 공백을 넣습니다.
목록의 모든 항목이 슬라이드에 들어가지 않으면 자동으로 스크롤됩니다.

* 순서 없는 목록 항목 1
* 순서 없는 목록 항목 2
* 순서 없는 목록 항목 3
----
#### 순서 있는 목록 (숫자)

순서 있는 목록에서는 항목에 자동으로 번호가 매겨집니다 🔢 .

1. 숫자 목록 항목 1
2. 숫자 목록 항목 2
3. 숫자 목록 항목 3
----
#### 순서 있는 목록 (알파벳)

각 항목에 알파벳 문자를 부여할 수도 있습니다 🔤 .

a. 알파벳 목록 항목 1
b. 알파벳 목록 항목 2
c. 알파벳 목록 항목 3
----
#### 표

Markdown 표준 구문으로 표를 만들 수 있습니다.
모든 줄은 파이프 문자로 시작하고 끝나야 합니다.

| 이름  | 점수 | 등급 |
|-------|------|------|
| Alice | 95   | A    |
| Bob   | 87   | B+   |
| Carol | 72   | C    |

### 텍스트 장식 및 주석
----
#### 텍스트 장식

텍스트를 굵게, 기울임꼴, 밑줄, 하이라이트로 꾸밀 수 있습니다.

* 이것은 *기울임꼴* 텍스트
* 이것은 **굵은** 텍스트
* 이것은 _밑줄_ 텍스트
* 이것은 ==하이라이트== 텍스트
----
#### 텍스트 노트

본문과 분리된 노트 📔 에 짧은 텍스트를 표시할 수 있습니다.
예를 들어, 이 문장이 하이라이트될 때 아래에 노트 텍스트가 표시됩니다.{note: 이것은 노트 텍스트입니다}
노트 텍스트는 마우스로 화면 위를 드래그할 수 있습니다.{note: 이 상자를 드래그해 보세요!}
----
#### 팝업 이미지

설명용 미니 이미지 🖼 를 추가할 수도 있습니다. {image: https://yohasebe.github.io/paradocs/img/sample/monalisa.jpg}
이미지는 적절한 크기로 조정되며 마우스로 드래그할 수 있습니다. {image: https://yohasebe.github.io/paradocs/img/sample/starrynight.jpg}
이미지를 확대하려면 이미지 위의 플러스 버튼을 클릭하세요. {image: https://yohasebe.github.io/paradocs/img/sample/greatwave.jpg}
----
#### 퀴즈

처음에는 청중에게 텍스트를 숨기고 적절한 시점에 공개하는 퀴즈 🙋 를 만들 수 있습니다.

| * 정적 텍스트 블록에서 중괄호 안의 단어와 구문은 {퀴즈 항목}이 됩니다
| * {불투명한 상자}로 덮여서 렌더링되며
| * 청중을 위한 {퀴즈를 만들} 수 있습니다
----
#### 객관식 퀴즈

클릭할 수 있는 객관식 퀴즈를 만들 수 있습니다.
정답 선택지 앞에 별표를 붙입니다.

| {mcq: 프랑스의 수도는 어디입니까?
|   a) 런던
|   b) 베를린
|   *c) 파리
|   d) 로마
| }

### 추가 기능
----
#### TTS 음성 읽기

일반 단락의 문장은 브라우저에서 사용 가능한 TTS 엔진 🗣 으로 음성 재생할 수 있습니다.
사용 가능한 언어와 음성 종류는 OS와 브라우저에 따라 다릅니다.

하이라이트된 콘텐츠에 TTS를 사용할 수 있을 때 프레젠테이션 화면 왼쪽 상단에 스피커 아이콘 🔈 이 나타납니다.
이 아이콘을 클릭하거나 마침표 키를 누르면 TTS 음성이 재생됩니다.
재생 중에는 현재 읽히고 있는 단어가 노란색 배경으로 하이라이트됩니다.

----
#### 자동 문장 분할

!! 자동 문장 분할 기능을 사용할 수 있습니다. 텍스트 앞에 느낌표 두 개를 붙이기만 하면 됩니다. 각 문장은 마침표나 다른 문장 부호로 끝나야 합니다. Paradocs가 텍스트를 자동으로 개별 문장으로 분할합니다.
----
#### 메모지(스티키 노트)

프레젠테이션 중 떠오르는 내용을 메모지에 작성하여 청중에게 보여줄 수 있습니다.

프레젠테이션 화면 왼쪽 상단의 메모지 아이콘 📃 을 클릭하여 메모지를 표시하거나 숨길 수 있습니다.
메모지의 내용은 슬라이드가 바뀌어도 유지됩니다.
----
#### 자동 프레젠테이션

완전 자동 프레젠테이션을 하려면 화면 왼쪽 상단의 마법 지팡이 아이콘 🧙 을 클릭하세요.
현재 또는 다음에 하이라이트될 프래그먼트부터 전체 프레젠테이션의 마지막 프래그먼트까지 텍스트 읽기, 동영상 재생 등이 자동으로 실행됩니다.
컴퓨터가 전체 프레젠테이션을 대신해 줄 수 있습니다 🤖 ❗
