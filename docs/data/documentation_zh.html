2. 基本概念

句子

句子之间通过单个换行符（↩）分隔。在段落中，句子被视为相邻元素，依次逐个高亮显示。 自动分句功能可以根据所使用的语言自动将文本拆分为句子。要执行自动分句，请在文本开头添加 !! 前缀。

段落

由多个句子组成的块称为段落。段落由多行文本组成，每行包含一个句子。

块

块之间通过两个或更多连续换行符 ↩↩ 分隔；一个块可以包含多个元素。由多个句子组成的块特别称为段落。

幻灯片

幻灯片也可以称为"页面"。幻灯片之间通过三个或更多连字符（---↩）分隔。一张幻灯片可以包含多个块。

如何编写段落

将多个句子垂直排列，每行一个句子，即可构成段落。

----
Sentence #1
Sentence #2
Sentence #3
----

使用自动分句功能，以上内容可以写成如下形式：

----
!! Sentence #1 Sentence #2 Sentence #3
----

要执行自动分句，请在文本开头添加 !! 前缀。同时，每个句子必须以句号或其他标点符号结尾。

如何在一张幻灯片中排列多个块

要在一张幻灯片上放置多个块，请在块之间留一个空行。

----
Block #1 Sentence #1
Block #1 Sentence #2

Block #2 Sentence #1
Block #2 Sentence #2
----

创建多张幻灯片（页面）

幻灯片也可以理解为页面。一张幻灯片可以包含多个块。

----
Slide #1
----
Slide #2
----
Slide #3
----

3. 块的类型

以下是对各种块类型的说明。

标题

您可以创建不同级别的标题。标题将以所选的强调色显示。

----
# Heading 1

## Heading 2

### Heading 3
----

普通段落

普通段落由每行一个句子的块构成。

This is regular paragraph.
Sentences in static text are highlighted when focused.
They are printed in grey when not focused.

在普通段落中，按键会将高亮从一个句子移动到下一个句子。当文本无法同时显示在幻灯片中时，幻灯片会自动滚动。

静态段落

要创建静态段落，请在构成段落的每个句子开头放置一个竖线符号和一个空格。

| This is static paragraph.
| Sentences in static text are not highlighted
| They are always printed in black.

或

| This is static paragraph.
  Sentences in static text are not highlighted
  They are always printed in black.

在查看静态段落时，逐句高亮功能会被关闭。需要注意的是，当静态段落内容超出幻灯片范围时，不会自动滚动。

无序列表

这就是所谓的项目符号格式。要描述无序列表的元素，请在每行开头放置一个星号和一个空格。

* Unordered list item 1
* Unordered list item 2
* Unordered list item 3

当所有元素无法在幻灯片中完全显示时，列表会自动滚动。

有序列表

这是顺序列表的格式。要描述此类列表的元素，请在每行开头放置一个数字/字母、一个句点和一个空格。

1. Ordered list item 1
2. Ordered list item 2
3. Ordered list item 3

a. Ordered list item 1
b. Ordered list item 2
c. Ordered list item 3

当所有元素无法在幻灯片中完全显示时，列表会自动滚动。

注意 无法指定列表的起始数字/字母。

编号块

您可以创建带有指定编号的块。这在演示中展示公式或练习题时非常有用。

365. The number in a numbered block will be printed only once;
     the left-hand side of the text can have multiple lines
     the lines are aligned nicely

注意 上述示例中第二行及后续行左侧的空格是必需的。

4. 文本装饰与注释

文本装饰

文本片段可以进行加粗、斜体、下划线或高亮处理。

Italic:    orange *strawberry* apple
Bold:      orange **strawberry** apple
Underline: orange _strawberry_ apple
Highlight: orange ==strawberry== apple

Markdown 兼容性

Paradocs 支持 Markdown 语法的一个子集，同时也有自己的格式规则。下表总结了支持的内容及其与标准 Markdown (GFM) 的区别。

功能 Paradocs 语法 标准 Markdown 备注
加粗 **text** **text** 语法相同
斜体 *text* *text* 语法相同
下划线 _text_ _text_（斜体） Paradocs 使用 _ 表示下划线，而非斜体
高亮 ==text== 不适用 Paradocs 扩展功能
删除线 ~~text~~ ~~text~~ 兼容 GFM
表格 | col | col | | col | col | 兼容 GFM；通过 marked.js 渲染
链接 [文本](url) [文本](url) 语法相同
图片 ![alt](url) ![alt](url) 语法相同；还支持 ![alt](local:文件名) 用于上传的图片
标题 # Heading # Heading 语法相同（支持 1-3 级）
无序列表 * item * item 语法相同
有序列表 1. item 1. item 语法相同；还支持 a. item
行内代码 `code` `code` 语法相同
引用块 > 文本 > 文本 语法相同
代码块 ``` ``` 围栏代码块；语言标识（如 ```python）将被忽略

注意： Paradocs 拥有专为演示用途设计的文本解析器。虽然部分语法与 Markdown 重叠，但请注意以下区别：

  • _text_ 产生下划线，而非斜体（斜体请使用 *text*）
  • ---（3个以上连字符）和====（4个以上等号）是幻灯片分隔符，而非水平线
  • 原始 HTML 标签（如 <b>、<div>）会被转义并显示为纯文本
  • 不支持嵌套列表（仅可使用扁平列表）
  • 不支持 Setext 样式标题（标题后跟 === 或 ---）；请使用 ATX 样式（# 标题）
  • 如需全幻灯片媒体展示，请使用 Paradocs 专用命令（image:、video: 等）

显示注释和弹出图片

可以在与正文分开的注释中显示简短文本。注释使用花括号和 note 命令创建。除了文本注释之外，您还可以使用 image 命令指定文件 URL 来创建图片弹窗。

This is part of main text. {note: This is a note}
This is also part of main text. {note: This is another note}
You can also add a pop-up image. {image: url_to_your_image_file.(png|jpg|gif)}

注意 注释和弹出图片可用于"普通段落"和"无序/有序列表"中（不适用于静态段落）。使用自动分句功能时无法显示注释和弹出图片。花括号 { } 及其内部的文本将被删除。

测验

使用花括号和 quiz 命令，您可以创建测验效果——文本初始处于隐藏状态，演示者在适当时机点击即可揭示答案。

| In a static context, words and phrases in {curly brackets}
| will be rendered as {quiz items} overlayed by an opaque box.

注意 创建测验时，必须使用静态段落或静态有序/无序列表。

选择题（MCQ）

您可以创建带有可点击选项的选择题。演示者点击选项后会显示正误结果。将 MCQ 块放在静态段落中（以 | 开头的行）。

| {mcq: What is the capital of France?
|   a) London
|   b) Berlin
|   *c) Paris
|   d) Rome
| }

在正确答案的选项字母前加上 * 进行标记。点击正确选项会以绿色高亮显示。点击错误选项只会将该选项以红色高亮，并显示重试提示——正确答案不会被揭示，用户可以继续尝试。回答正确后会出现一个重试按钮，允许重置测验并再次作答。

表格

您可以使用标准 Markdown 表格语法创建表格。所有行必须以 | 开始和结束，第二行必须是分隔符。

| Name  | Score |
|-------|-------|
| Alice | 95    |
| Bob   | 87    |

注意 以 | 开始和结束的表格行与仅在行首有 | 的静态段落是有区别的。

5. 嵌入媒体文件

您可以在幻灯片中嵌入图片、视频和音频。

嵌入图片

您可以指定上传到网络上的 PNG/JPG/GIF 格式图片文件的 URL，并将其显示在幻灯片上。图片会被尽可能放大显示。

image: url_to_your_image_file.png

或

img: url_to_your_image_file.png

注意 要在幻灯片上显示图片，请确保上述命令是该幻灯片上唯一的块。如果存在多个块，图片本身不会显示，而是以图片链接的形式呈现。

提示

如果您想嵌入自己的图表，可以使用 Google Drawings，并粘贴其"发布到网络"功能生成的图片链接。您也可以使用 Mermaid Live Editor，获取"Link to SVG"来嵌入流程图或甘特图等商业图表。

本地图片

您可以使用编辑器旁边的拖放区域从设备上传图片。支持的格式：JPEG、PNG、GIF、WebP（每张最大 5MB）。上传的图片保存在浏览器的本地存储中，页面重新加载或浏览器重启后仍然保留。点击全部重置按钮或手动清除浏览器数据后将被删除。

要将本地图片插入幻灯片，请点击图片列表中的文件名，或输入：

image: local:your_image_file.jpg

您也可以使用标准 Markdown 图片语法来插入行内本地图片：

![描述](local:your_image_file.jpg)

嵌入 YouTube 视频

您可以指定 YouTube 视频的 URL 并将其显示在幻灯片上。视频会被尽可能放大显示。

youtube: https://www.youtube.com/watch?v=Ks-_Mh1QhMc

或

yt: https://www.youtube.com/watch?v=Ks-_Mh1QhMc

您可以在 YouTube 视频的 URL 中指定起止时间点。可以通过视频开头起经过的秒数来指定。以 &start=x&end=y 格式将选项添加到 URL。以下示例将播放视频的第 30 秒（x=30）到第 60 秒（y=60）。

youtube: https://www.youtube.com/watch?v=MMmOLN5zBLY&start=30&end=60

您也可以使用"时:分:秒"格式指定起止时间，这对许多用户来说可能更加直观。

youtube: https://www.youtube.com/watch?v=MMmOLN5zBLY&start=0:30&end=1:00

注意 要在幻灯片上显示 YouTube 视频，请确保上述命令是该幻灯片上唯一的块。如果存在多个块，视频本身不会显示，而是以视频链接的形式呈现。

嵌入 MP4 视频

您可以指定上传到网络上的 MP4 格式视频文件的 URL，并将其显示在幻灯片上。视频会被放大至（接近）整个幻灯片页面。

video: url_to_your_video_file.mp4

您可以在视频的 URL 中指定起止时间点。请注意，每个时间点以视频开头起经过的秒数表示。以 #t=x,y 格式将选项添加到 URL。以下示例将从视频最开始播放（x=0），并在 5 秒后停止（y=5）。

video: url_to_your_video_file.mp4#t=0,5

注意 要在幻灯片上显示 MP4 视频，请确保上述命令是该幻灯片上唯一的块。如果存在多个块，视频本身不会显示，而是以视频链接的形式呈现。

嵌入 MP3 音频

您可以指定上传到网络上的 MP3 格式音频文件的 URL，并将其显示在幻灯片上。

audio: url_to_your_audio_file.mp3

您可以在 MP3 音频的 URL 中指定起止时间点。请注意，每个时间点以音频开头起经过的秒数表示。以 #t=x,y 格式将选项添加到 URL。以下示例将从音频最开始播放（x=0），并在 5 秒后停止（y=5）。

audio: url_to_your_audio_file.mp3#t=0,5

注意 单张幻灯片上可以包含多个音频文件。当所有块无法在当前视口中完全显示时，幻灯片页面会自动滚动。

6. 附加功能

文本转语音

普通段落中的句子可以通过浏览器内置的 Web Speech API（speechSynthesis）进行朗读。这是浏览器的标准 API，不限于 Chrome，在 Safari、Firefox、Edge 等现代浏览器中均可使用。

可用的语音包括浏览器提供的语音（如 Chrome 中的 Google TTS、Edge 中的 Microsoft TTS）以及操作系统级别的系统语音（如 macOS 的 Siri 语音、Windows 的 SAPI 语音）。您可以在输入页面的下拉菜单中选择语音合成语言和语音合成声音。语音速度也可以调节，但部分语音可能不支持速度更改。

当高亮内容支持 TTS 时，演示屏幕左上角会出现一个扬声器图标。点击此图标或按键盘上的 . 键即可播放 TTS 语音。再次点击按钮或按键即可停止播放。

在 TTS 播放过程中，当前正在朗读的单词会以黄色背景高亮显示。这种单词级高亮可以帮助观众跟随朗读内容。播放结束或停止时，高亮会自动清除。

注意 单词级高亮依赖于浏览器的 onboundary 事件支持。在 Chrome 和 Edge 中效果良好，但并非所有浏览器都支持此功能。如果操作系统安装了其他语言包，这些语音也会出现在语音列表中。

云端TTS（高级）

除了浏览器内置的 TTS 外，Paradocs 还支持通过 OpenAI 和 ElevenLabs API 进行基于云端的文本转语音。云端 TTS 可提供更高质量、更自然的语音。

要使用云端 TTS，请展开输入页面上的**云端TTS（高级）**部分：

  1. TTS 提供商：从下拉菜单中选择 OpenAI 或 ElevenLabs。选择云端提供商后，浏览器 TTS 控件将隐藏，显示云端专用选项。
  2. API 密钥：输入所选提供商的 API 密钥。点击 Verify 按钮检查密钥是否有效。您的密钥仅保存在浏览器的本地存储中，不会发送到提供商 API 以外的任何服务器。
  3. 语音：对于 OpenAI，可从固定的语音列表（alloy、ash、ballad、coral、echo、fable、nova、onyx、sage、shimmer）中选择。对于 ElevenLabs，输入有效的 API 密钥后会自动从您的账户获取语音列表。
  4. 速度（仅限 OpenAI）：可在 0.5 倍至 2.0 倍的范围内调整播放速度。

限制：

  • 云端 TTS 不支持播放过程中的单词级高亮（云端生成的音频无法触发浏览器的 onboundary 事件）。
  • 需要有效的网络连接和 API 密钥。
  • API 使用可能会根据提供商的定价产生费用。
  • 导出为独立 HTML 文件时，出于安全考虑，API 密钥不会包含在导出文件中。导出文件中将无法使用云端 TTS。

便签

您可以在演示过程中将想到的内容写在便签上，展示给观众。

点击演示屏幕左上角的便签图标即可显示或隐藏便签。即使幻灯片切换，便签的内容也会保留。

激光笔

您可以将鼠标指针从普通样式切换为类似激光笔的彩色圆点。点击屏幕左上角的 ● 图标即可切换模式。

鼠标指针的颜色由设置面板中"强调色"的选择决定。

自动演示

点击屏幕左上角的魔法棒图标。从当前幻灯片的下一个片段到整个演示文稿的最后一个片段，文本朗读、视频播放等将自动依次执行。

自动保存

输入的文本、表单设置和上传的本地图片会自动保存到浏览器的本地存储中。即使关闭浏览器，再次访问页面时之前的工作内容也会自动恢复。点击清除文本仅清除编辑器文本，点击全部重置可清除文本、设置和所有上传的图片。

文件菜单（保存 / 加载 / 下载）

文件下拉按钮提供文件操作：

  • 保存文本 — 将编辑器当前文本下载为 .txt 文件（paradocs-source.txt）。
  • 加载文本 — 打开文件选择器，将 .txt 或 .md 文件加载到编辑器中。文件类型（仅文本）、大小（最大 1MB）和内容安全性（script 标签和事件处理器将被去除）会被验证。
  • 下载HTML — 将演示文稿导出为独立的 HTML 文件（参见下文）。

下载 HTML

点击文件菜单中的下载HTML，将您的演示文稿导出为独立的 HTML 文件。该文件包含所有幻灯片内容、配置和样式，因此无需访问 Paradocs 网站即可在任何浏览器中打开。外部库依赖（jQuery、Reveal.js 等）通过 CDN 加载。

注意 在导出的文件中，YouTube 视频嵌入会自动转换为可点击的缩略图链接，因为由于浏览器安全限制，本地 HTML 文件无法嵌入 YouTube iframe。点击缩略图即可在 YouTube 上打开视频。

幻灯片位置与内部链接

演示文稿的 URL 会在您浏览幻灯片和片段时自动更新（例如 #/0/1/3）。您可以利用这些基于哈希的 URL 创建内部链接，在演示文稿内跳转到特定幻灯片——例如使用锚标签 <a href="#/2">跳转到第3张幻灯片</a> 从一张幻灯片链接到另一张。

胶片条预览

点击操作按钮旁边的预览按钮（眼睛图标），即可在编辑器右侧打开胶片条面板。面板会显示幻灯片缩略图，在您输入文本时自动更新（延迟800毫秒）。

  • 同步开启时（默认）：单击缩略图可将编辑器光标跳转到对应幻灯片文本并选中。
  • 同步关闭时：单击缩略图可在灯箱中放大显示幻灯片。使用方向键在幻灯片间导航，按 Escape 或点击外部关闭。
  • 编辑器标题栏中的同步开关（链接图标）可启用胶片条自动滚动以跟随光标位置。
  • 拖动编辑器与面板之间的调整手柄可改变胶片条宽度。

如果输入过程中发生解析错误，预览会保留上次成功渲染的内容。

注意 胶片条面板仅在浏览器窗口宽度超过 900px 时可用。在较小的屏幕上，预览按钮会自动隐藏。胶片条采用虚拟滚动技术，即使幻灯片数量很多也能保持较低的内存使用量。

概览模式

在演示过程中按 ESC 键可切换概览模式。所有幻灯片会以网格形式的缩略图展示，便于快速浏览大型演示文稿。

  • 使用方向键（←→↑↓）在幻灯片之间移动选择。
  • 按 Enter 跳转到所选幻灯片并返回演示。
  • 再次按 ESC 关闭概览并返回当前幻灯片。
  • 单击任意缩略图可直接跳转到该幻灯片。

YouTube 视频幻灯片会显示缩略图预览而非嵌入播放器。演示屏幕右上角的网格图标也可以切换概览模式。

提示音

点击演示屏幕左侧工具栏的铃铛图标，可在每次推进一个片段（句子或列表项）时播放一声短促的提示音。这有助于让观众注意到演示中的每个步骤。再次点击图标可禁用提示音。

反色

勾选反色复选框可反转高亮部分的显示方式。默认情况下，高亮文本为彩色文字配透明背景；反转后，高亮颜色将作为填充背景，文字变为白色。
